TomsTools

  • 在资源管理器中找到所选文件或文件夹
  • 复制所选（支持多个）文件或文件夹的路径（可通过配置去除路径中的指定前缀）
  • 打开命令行窗口，并切换到所选文件或文件夹所在目录
  • 如果是properties文件，则可将其国际化为指定语言后缀（默认zh_CN）的资源文件
  • 所有功能可在Eclipse属性页（preferences）进行配置
  • 支持多语言。该插件在Eclipse属性页和右键的显示名称是：“Tom工具箱”，英文名为“TomsTools”
  • 集成java类搜索功能，能够在指定目录（含jar包）中搜索指定的java类。
